Python传参机制

Python 中一切皆为对象，数字是对象，列表是对象，函数也是对象，任何东西都是对象。而变量只是对象的一个引用，对象的操作都是通过引用来完成的

个人认为，python函数内重新赋值时，不会改变函数外的值，但在函数内修改成员变量的值时，会改变函数外的值
